# Ci360 Event Download Agent



## Getting started

This tool allows you to download and save CI360 events into a database through a CI360 Custom Agent.  The full event information is stored as a JSON data type. Simple ready to use package with minimal implementation steps.

## Requirements

- JDK 17 or JDK 21
- Any database that supports/provides a JDBC driver and supports a JSON data type
- JDBC Driver software for the database
